# Movie Genre Classification

This project is part of the CodSoft Machine Learning Internship.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Logistic Regression
- NLP

## Description
This machine learning project predicts movie genres based on movie plot descriptions using Natural Language Processing and TF-IDF Vectorization.

## Features
- Text preprocessing and cleaning
- Stop-word removal
- TF-IDF feature extraction
- Genre prediction using Logistic Regression
- Classification report and accuracy evaluation

## Output
The model predicts genres such as comedy, drama, thriller, action, etc.
